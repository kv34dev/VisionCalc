## Tech Stack

- **Language**: Swift
- **Framework**: SwiftUI
- **Architecture**: MVVM-like structure
- **Platform**: Apple Vision Pro (Xcode project)

## Installation

1. Clone this repository:
```
git clone https://github.com/kb34dev/visioncalc
```
2. Navigate to the project directory:
```
cd visioncalc
```
3. Open the project in Xcode:
```
open visioncalc.xcodeproj
```
4. Build and run the app on a Vision Pro simulator or device.
